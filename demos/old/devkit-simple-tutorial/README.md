## About

This project only includes the Web App base for the step-by-step introductory tutorial.

Check out the instructions online [on blindnet.dev](https://blindnet.dev/docs/devkit-tutorial).

## Dev Tools

This Web App demo is built with [open-wc recommendations](https://github.com/open-wc).

Use [classic Yarn (v1)](https://classic.yarnpkg.com/en/docs/install#debian-stable) (`yarn <script-name>`) to run the following scripts:

- `start` runs your app for development, reloading on file changes
- `start:build` runs your app after it has been built using the build command
- `build` builds your app and outputs it in your `dist` directory
- `test` runs your test suite with Web Test Runner
- `lint` runs the linter for your project
- `format` fixes linting and formatting errors
